**Лабораторная работа по линейной алгебре "Сравнение методов решения СЛАУ"**  

Набор функций для решения систем линейных алгебраических уравнений (СЛАУ) методом LU-разложения и методом QR-разложения и проведения экспериментов по сравнению эффективеости и точности методов.

**Инструкция по сборке (команды):**
* make -- компиляция, запуск первого эксперимента по умолчанию
* make exp1 -- компиляция, запуск первого эксперимента
* make exp2 -- компиляция, запуск второго эксперимента
* make exp3 -- компиляция, запуск третьего эксперимента
* make clean -- очистка сгенерированных файлов
